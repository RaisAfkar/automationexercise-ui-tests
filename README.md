# 🧪 AutomationExercise UI Testing

This repository contains an automated UI testing suite for [AutomationExercise.com](https://automationexercise.com), built using **Python**, **Selenium WebDriver**, and **Pytest** framework.

### Tech Stack
- **Language:** Python 3
- **Framework:** Pytest
- **Automation Tool:** Selenium
- **Reporting:** Allure Report
- **Test Coverage:**
  - User Registration
  - Login / Logout
  - Add to Cart & Checkout

### Purpose
To ensure core user journeys on AutomationExercise.com are functioning as expected through continuous UI regression testing.
